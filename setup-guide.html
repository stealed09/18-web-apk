🛠️ Developer Setup Guide

One-time setup to enable client to update APK via Google Sheets

1

Create Google Sheet

  1. Go to sheets.google.com
  2. Create a new blank spreadsheet
  3. Name it "APK Config"
2

Add Headers & Data

Fill in the sheet exactly like this:

apk_link
version
size
updated
https://github.com/...
v1.0
24.5 MB
Jan 2025

Row 1: Headers (apk_link, version, size, updated)

Row 2: Actual values (this is what client will edit)

3

Make Sheet Public

  1. Click Share button (top right)
  2. Under "General access", select "Anyone with the link"
  3. Set role to "Viewer" (we only need read access)
  4. Click Done

⚠️ Important

If you want client to EDIT the sheet, set role to "Editor". Otherwise they can only view but not change.

4

Get Sheet ID

Look at your browser URL. It will look like:

https://docs.google.com/spreadsheets/d/1A2B3C4D5E6F7G8H9I0J/edit

The highlighted part is your SHEET_ID. Copy it!

✅ Example Sheet IDs

1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms
1a2b3c4d5e6f7g8h9i0jklmnopqrstuvwxyz

5

Paste Sheet ID in Code

Open index.html and find this line (around line 1041):

const SHEET_ID = "YOUR_SHEET_ID_HERE";

Replace YOUR_SHEET_ID_HERE with your actual Sheet ID:

const SHEET_ID = "1A2B3C4D5E6F7G8H9I0J";
6

Deploy to Netlify

  1. Save index.html
  2. Drag & drop the file/folder to Netlify
  3. Your site is live!
7

Share Sheet Link with Client

Send the client the Google Sheet edit link:

https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit

Also send them guide.html — it has simple visual instructions in Hindi!

🎉 Setup Complete!

Now client can simply:
Open Sheet → Paste new APK link in A2 → Done!
Website automatically fetches latest link every time it loads.

📊 How it Works (Technical)

  • Google Sheets can be accessed as JSON via special URL
  • Website fetches /gviz/tq?tqx=out:json endpoint
  • Parses row 2 values and updates CONFIG object
  • Download button uses CONFIG.APK_LINK
  • Has fallback values if fetch fails
Setup once, update forever ❤️
